## Getting Started

### Dependencies

* Python 3
* Pygame
* Numpy
* Keras
* Tensorflow

### Play Gomoku

#### This command runs a program where you can play gomoku with a friend.
```
python gomoku.py
```

#### This command runs a program where you can play gomoku against an AI opponent.
```
python gomoku_with_AI.py
```

#### This command runs a program where two AI play Gomoku against each other.
```
python two_ai_play_gomoku.py
```
